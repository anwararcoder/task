## Setup Project & Run Project
1- Clone This Repo :
#### `https://github.com/anwararcoder/task.git`

2- Install NPM Package
#### `npm install`

3- Run NPM Package
#### `npm start`

4- Run Database Server
#### `npm run server` Or Open http://localhost:5000/login Or http://localhost:5000/register

3- Open Project
#### `http://localhost:3000`
